# Itens de Mão

Itens de mão são todos os objetos que podem ser empunhados, carregados ou utilizados diretamente pelas mãos do personagem, seja em combate, exploração, interação social ou mágica. Esta categoria inclui armas, escudos, ferramentas, instrumentos e outros objetos funcionais.

## Tipos de Armas

- Arcos
- Bestas
- Espadas
- Maças
- Lâminas curtas

## Modelo de Ficha de Tipo de Arma

- Nome
- Tipo (corpo, à distância, cerimonial)
- Dano base
- Propriedades (falcão, pesado, leve, duas mãos)
- Requisitos de uso (força, perícia)

## Modelo de Ficha de Armas

Campos sugeridos para cada arma específica:

- Nome
- Descrição
- Dano
- Peso
- Valor
- Raridade
- Efeitos especiais
